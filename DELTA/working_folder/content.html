No estudo das séries numéricas, um dos tópicos mais importantes é a determinação de sua convergência. Em diversas situações, precisamos entender quando uma série infinita converge para um valor finito ou quando ela diverge, isto é, quando a soma dos seus termos não chega a um número bem definido. Um dos métodos mais usados para estudar a convergência das séries é o Teste da Razão, que é eficiente para séries cujos termos se relacionam de maneira simples entre si. Seja \((a_n)_n\) uma sequência real. Suponha que existe o limite \[ L = \lim_{n \to \infty} \frac{|a_{n+1}|}{|a_n|} \] Então: Se \(L < 1\), a série \(\sum_n a_n\) converge absolutamente. Se \(L > 1\) (ou \(L = +\infty\)), a série diverge. Se \(L = 1\), o teste é inconclusivo. Esse teorema é extremamente útil quando tentamos estudar séries que possuem um comportamento assintótico simples, onde a razão entre os termos consecutivos pode ser facilmente calculada. O Teste da Razão fornece uma maneira direta de tomar uma decisão sobre a convergência ou divergência das séries, mas em alguns casos, quando L é igual a 1, precisamos de testes adicionais para obter uma resposta conclusiva.
